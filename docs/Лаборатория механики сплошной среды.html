Лаборатория UniCFD

Web–лаборатория UniCFD создана на базе технологической платформы UniHUB и предназначена для поддержки открытого сообщества исследователей, чьи научные интересы связаны с методами решения задач механики сплошной среды с использованием свободных прикладных пакетов. Лаборатория открыта для новых участников.

Основными направлениями деятельности в рамках лаборатории являются: образование, консалтинг, разработка программного обеспечения, поверочные расчёты, моделирование, проектирование конструкций и изделий, фундаментальные исследования.
